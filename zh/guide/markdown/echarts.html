            让你 VuePress 站点中的 Markdown 文件支持图表。

            此插件使用 EChartsopen in new window 提供相应功能。

            # 配置

            # 格式

            ::: echarts 标题

```json
{
  // 此处为 ECharts 图表配置
}
```

:::

            我们也支持 js 和 javascript 的代码块，你应当将导出对象赋值给 module.exports。

            # 案例

            # 折线图

            一个折线图案例
            代码
            ::: echarts 一个折线图案例

```json
{
  "xAxis": {
    "type": "category",
    "data": ["Mon", "Tue", "Wed", "Thu", "Fri", "Sat", "Sun"]
  },
  "yAxis": {
    "type": "value"
  },
  "series": [
    {
      "data": [150, 230, 224, 218, 135, 147, 260],
      "type": "line"
    }
  ]
}
```

:::

            # 柱状图

            一个柱状图案例
            代码
            ::: echarts 一个柱状图案例

```json
{
  "xAxis": {
    "type": "category",
    "data": ["Mon", "Tue", "Wed", "Thu", "Fri", "Sat", "Sun"]
  },
  "yAxis": {
    "type": "value"
  },
  "series": [
    {
      "data": [120, 200, 150, 80, 70, 110, 130],
      "type": "bar",
      "showBackground": true,
      "backgroundStyle": {
        "color": "rgba(180, 180, 180, 0.2)"
      }
    }
  ]
}
```

:::
