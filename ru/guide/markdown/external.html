    # Markmap

    Markmap — это инструмент для преобразования Markdown в интеллект-карты. Он поддерживает больше форматов и контента, чем встроенная блок-схема темы.

    1. Используйте Markmapopen in new window для создания HTML-файла интеллект-карты
    2. Поместите файл HTML в папку .vuepress/public/
    3. Вставить в Markdown через <iframe>

    Ввод:

    <iframe
  :src="$withBase('/markmap/demo.html')"
  width="100%"
  height="400"
  frameborder="0"
  scrolling="No"
  leftmargin="0"
  topmargin="0"
/>
